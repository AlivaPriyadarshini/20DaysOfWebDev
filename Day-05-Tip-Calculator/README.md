# 💰 Tip Calculator

A simple and responsive **Tip Calculator** built using **HTML, CSS, and JavaScript**. This project calculates the tip amount and total bill based on the entered bill amount and tip percentage, providing instant results with a clean and modern user interface.

## 🚀 Features

* 💵 Calculate tip amount instantly
* 🧮 Calculate total bill including tip
* 📝 Input validation for empty fields
* ⚡ Real-time calculation on button click
* 🎨 Modern glassmorphism UI
* 📱 Responsive and beginner-friendly design
